### Install
Clone this project

```bash
> git clone https://github.com/dvnz99/wabot
> cd wabot
```

Install the dependencies:

```bash
> npm install
> npm install -g sharp
> npm install -g pm2
```

### Usage
**1. run the Whatsapp bot**

```bash
> npm start
or 
> node .
```
or
```bash
> pm2 start .
> pm2 monit
```

**2. stop the Whatsapp bot**

for npm :
```bash
> ctrl + c
```
for pm2 :
```bash
> pm2 stop
```

after running it you need to scan the qr
